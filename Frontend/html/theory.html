HTML

1.1 HTML Structure

The structure of an HTML document is the foundation of any webpage. It defines how the content is organized and presented in a browser.

  1. <!DOCTYPE> Declaration: This declaration defines the document type and version of HTML. It ensures that the browser renders the page in standards-compliant mode.
  2. <html> Element: This is the root element of the HTML document. All other elements are nested within it.
  3. <head> Element: This element contains meta-information about the document, such as the title, character set, and links to stylesheets or scripts. It doesn't contain visible content.
  4. <body> Element: The <body> element contains all the visible content of the webpage, such as text, images, links, and more.

    5. Structure :

     
    
              <!DOCTYPE html>
              <html lang="en">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title> My First Webpage </title>
                </head>
                <body>
                  <h1> Welcome to My Webpage </h1>
                  <p> This is a paragraph of text on my first webpage. </p>
                </body>
              </html>

    1.2 HTML Common Tags

    1. <p> Tag (Paragraph): The <p> tag is used to define a paragraph of text. It's a block-level element, meaning it starts on a new line and takes up the full width available.
      Example: 
      
                <p>This is a paragraph of text.</p>

    2. <h1> Tag (Heading): The <h1> tag is used for the main heading of a webpage. It's the largest and most important heading, typically used once per page to define the main title or topic.
      Example: 
      
                <h1>Welcome to My Website</h1>
      Explanation: This tag represents the top-level heading, making the text bold and large. There are also <h2>, <h3>, etc., for subheadings with decreasing importance.

    3. <a> Tag (Anchor/Link): The <a> tag is used to create hyperlinks, which can link to other pages, files, or locations within the same page. Example: 
      
              <a href="https://www.pvgcoet.ac.in/">PVG COET</a>
      Explanation: The href attribute specifies the URL of the link's destination. The text between the opening and closing <a> tags becomes the clickable part of the link.

    1.3 HTML Lists

    1. <ul> Tag (Unordered List): The <ul> tag is used to create an unordered list, where items are typically marked with bullet points. The order of the items doesn't matter.
      Example: 
      
                <ul>
                  <li>Mango</li>
                  <li>Apple</li>
                  <li>Banana</li>
                </ul>
      Explanation: The <ul> tag wraps the entire list, and each item within the list is wrapped in an <li> (list item) tag. By default, each item is displayed with a bullet point.

    2. <ol> Tag (Ordered List): The <ol> tag is used to create an ordered list, where items are marked with numbers or letters, indicating that the order is important.
      Example: 
      
                <ol>
                  <li>One</li>
                  <li>Two</li>
                  <li>Three</li>
                </ol>

    1.4 HTML Forms

    1. <form> Tag: The <form> tag is used to create an HTML form for user input. It acts as a container for various form elements like text fields, checkboxes, radio buttons, and submit buttons.

      Attributes:
      action: Specifies where the form data should be sent when submitted (e.g., a server script).
      method: Defines the HTTP method (GET or POST) to be used when sending form data.

      Example: 
      
                <form action="/submit_form.php" method="POST">
                    <!-- Form elements go here -->
                </form>
      Explanation: This tag wraps all the input elements of a form. The action attribute determines the server endpoint where the data is sent, and the method attribute specifies how the data is sent.

    2. <input> Tag: The <input> tag is used to create various types of input fields. It's a self-closing tag that can take many forms, such as text boxes, radio buttons, checkboxes, etc., depending on the type attribute.

      Attributes:
      type: Specifies the type of input (text, password, email, radio, checkbox, etc.).
      name: Assigns a name to the input field, which is used to identify the data when the form is submitted.
      value: Specifies the default value of the input field.

      Example: 
      
                <input type="text" name="username" placeholder="Enter your username">
      Explanation: This creates a text input field for the user to enter their username. The placeholder attribute provides a hint to the user of what should be entered.

    3. <button> Tag The <button> tag is used to create a clickable button. It can be used to submit the form or trigger a JavaScript function.

      Attributes:
      type: Defines the button's behavior. Common values are 'submit', 'reset', or 'button'.

      Example: 
      
                <button type="submit">Submit</button>
      Explanation: This creates a button that, when clicked, submits the form data to the server as defined by the form's action attribute.


    Combined Code Example: 

    
            <!DOCTYPE html>
              <html lang="en">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport" content="width=device-width, initial-scale=1.0">
                  <title>HTML Form Example</title>
                </head>
                <body>
                  <h1>User Registration</h1>
                    <form action="/submit_registration.php" method="POST">
                    <label for="username">Username:</label>
                    <input type="text" id="username" name="username" placeholder="Enter your username">
                    <label for="password">Password:</label>
                    <input type="password" id="password" name="password" placeholder="Enter your password">
                    <button type="submit">Register</button>
                    </form>
                </body>
              </html>
